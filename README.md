# Netflix Content Strategy and Catalogue Analysis using Python

A Python portfolio project exploring how Netflix’s catalogue has evolved over time through content growth, format balance, geographic concentration, category structure, and recurring description themes.

## Project Summary

This project analyses a Netflix titles dataset to understand whether the catalogue’s apparent breadth reflects broad diversity or a more concentrated underlying structure.

The work focuses on:
- catalogue growth over time
- movies vs TV shows balance
- country concentration
- category concentration
- contributor recurrence
- text based theme patterns

This repository is intended as a portfolio case study showing practical skills in Python data cleaning, exploratory analysis, feature engineering, and insight communication.

## Project Files

- `notebooks/` – main Jupyter notebook containing the full analysis
- `report/` – designed PDF report summarising the project findings
- `data/` – raw and cleaned datasets
- `outputs/` – exported charts and visual assets
